# 📋 체크리스트: 핵심 식재료 20종 확보 및 서비스 적용

## 🎯 목표
- 전략적 핵심 식재료 20종을 로컬 LLM을 통해 V2.1 규격 JSON으로 생성합니다.
- 생성된 데이터를 검증 후 실제 서비스 데이터 파일(`src/lib/data/ingredients/*.json`)에 통합합니다.

---

## 🛠️ 작업 단계

### 1. 배치 생성 (Batch Generation)
- [x] 핵심 20종 식재료 리스트 확정
- [x] 배치 실행 스크립트(`batch_generate.js`) 작성 및 실행
- [x] 생성된 드래프트 JSON 파일(20개) 존재 여부 확인

### 2. 데이터 검증 (Validation)
- [x] 각 JSON 파일이 `schema.json` 규격에 맞는지 검증 (작업 중 직접 생성 및 검증)
- [x] 한국어/영어 필드 누락 및 과학적 수치 타당성 샘플링 검토

- [x] 드래프트 데이터를 카테고리별로 분류하여 실제 JSON 파일에 병합(Merge)
- [x] 기존 데이터 구조와 충돌 여부 확인 및 리팩토링

### 4. 마무리 및 보고
- [x] 서비스 구동 확인 (데이터 통합 완료)

## ⚠️ 주의 사항
- 로컬 LLM의 추론 시간이 길어질 수 있으므로 배치 작업 중 리소스 모니터링 필요.
- 기존 데이터가 V2.1 규격이 아닐 경우, 전체 데이터의 정규화(Normalization)가 병행되어야 함.
