# 💡 CONCEPT LOGIC: 핵심 가치 및 기획 의도

## 1. MVP 단계 핵심 로직 우선순위
1.  **Trust Logic (P0)**: 140종 식재료의 6대 풍미 데이터의 정합성. (완료)
2.  **Accuracy Logic (P0)**: 대체재별 고유 비율(`ratio`)에 기반한 정밀 계산. (완료)
3.  **Visual Logic (P1)**: 레이더 차트를 통한 '풍미의 상대적 차이' 시각화. (완료)
4.  **Context Logic (P1)**: '어떤 요리에 쓰나?'에 대한 활용 요리 데이터 제공. (완료)

## 2. 향후 보강될 개념 로직 (Next Steps)
- **Visual Evidence (P2)**: 텍스트 위주의 정보를 보완하기 위한 실물 이미지 매칭 로직.
- **Pairing Engine (P2)**: "이 재료를 쓸 때 함께 쓰면 시너지가 나는 재료" 추천 알고리즘.
- **Alternative Ranking**: 같은 재료에 대해 여러 대체재가 있을 경우, 용도별(볶음용, 국물용 등) 최적의 순위 제공.
